Platform for clinicians and patients to generate personalized home exercise programs using AI (Claude API).

Key domain concepts:
- Two roles: clinician, patient
- Exercise Library (curated, not AI-invented)
- AI workout generation constrained to library exercises
- Progression/regression chains for exercises
- Adherence tracking, outcome tracking, feedback loops
- Patient-clinician messaging
- HIPAA considerations for health data

**Why:** Anchors all architectural decisions to the actual product being built.
**How to apply:** Every technical recommendation must trace back to a concrete product requirement listed here.
